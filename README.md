# Userstyle installieren
 
Dieses Repo enthält ein CSS-Userstyle für die Browser-Extension **[Stylus](https://github.com/openstyles/stylus)**.
 
## Voraussetzungen
 
- **Stylus** installiert haben — verfügbar für [Firefox](https://addons.mozilla.org/firefox/addon/styl-us/), [Chrome](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne) und andere Chromium-Browser.
 
## Installation
 
1. Stylus öffnen → **Neuen Style erstellen**
2. Den **gesamten Inhalt** der `docbee.css`-Datei kopieren — inklusive der `@-moz-document`-Flags und dem Header oben
3. In den Stylus-Editor einfügen
4. **Speichern** (Strg + S)
 
> **Hinweis:** Die Flags am Anfang der Datei (z.B. `@-moz-document url-prefix(...)`) bestimmen, auf welchen Seiten der Style aktiv ist. Die müssen mit kopiert werden, sonst greift der Style nicht.
 
## Deinstallation
 
In Stylus den Style suchen → Löschen.
